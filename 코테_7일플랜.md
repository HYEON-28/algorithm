## 딱 3가지만 기억

1. **Day 3~4(新・Bランクレベルアップ 22문)에 제일 많은 시간.** "대부분의 B랭크 문제에 대응"되도록 설계된 가성비 최고 세트.
2. **막히면 30분 넘기지 말고 해설/정답 코드 보고 넘어가기.** 7일뿐 → "이해 후 다음"이 정답.
3. **답만 제출 = 브루트포스 적극 활용.** 효율 신경 끄고 "되는 코드" 먼저. 속도가 점수다 (60분 10문제 ≈ 문제당 6분).

## 실전 메모

- 메뉴 페이지에서 **언어 Java 선택**, 브라우저 번역 켜고 풀기 (실제 시험도 일본어일 가능성 높음 → 적응 연습).
- 무료로 B랭크 연습 충분. 굳이 유료 결제 X.
- 시험 중 화면·얼굴 녹화 필수 → 조용한 환경 + 캠 준비.
